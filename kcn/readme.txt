Show a random kannada verse from Chanakya Niti written by Chanakya on your website!

== Description ==

Chanakya Neeti is a collection of verse, said to be selected by Chanakya from the various shastras.
Its a Random collection of verse from Chanakya Neeti will be displayed.

== Installation ==

1. Upload the `kcn` folder to the `/wp-content/plugins/` directory
2. Activate the Kannada Chanakya Neeti plugin through the 'Plugins' menu in WordPress
3. Go to Appearance --> Widgets to add the Kannada Chanakya Neeti widget to any widgetized areas, such as sidebars.

== Frequently Asked Questions ==

= How many verses are there? =
There are actually 1000s of verse in  Chanakya Neeti. But this plugin has just over 25 verse. I'll add more verse in the future updates.

= How to style the widget? =
The verse output wrapped in `<p>` tag with `kcn` ID and class. Use `#kcn` or `.kcn` in your Custom CSS to style it.

= Can I add more verses to this plugin? =
Yeah, sure! You can add more verses to this plugin. Just open `kcn.txt` and add new verse at the bottom of the file. If you send me the updated file, I'll upload it to wordpress plugin repository.
