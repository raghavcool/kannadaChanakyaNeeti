Show a random kannada verse from Chanakya Niti written by Chanakya on your website!

== Description ==

Chanakya Neeti is a collection of aphorisms, said to be selected by Chanakya from the various shastras.
Its a Random collection of aphorisms from Chanakya Neeti will be displayed.

== Installation ==

1. Upload the `kcn` folder to the `/wp-content/plugins/` directory
2. Activate the Kannada Chankya Neeti plugin through the 'Plugins' menu in WordPress
3. Go to Appearance --> Widgets to add the Kannada Chankya Neeti widget to any widgetized areas, such as sidebars.

== Frequently Asked Questions ==

= How many poems are there? =
There are actually 1000s of aphorisms in  Chankya Neeti. But this plugin has just over 200 aphorisms. I'll add more aphorisms in the future updates.

= How to style the widget? =
The poem output wrapped in `<p>` tag with `kcn` ID and class. Use `#kcn` or `.kcn` in your Custom CSS to style it.

= Can I add more poems to this plugin? =
Yeah, sure! You can add more poems to this plugin. Just open `kcn.txt` and add new aphorisms at the bottom of the file. If you send me the updated file, I'll upload it to wordpress plugin repository.
